# 불임 분류 프로젝트 실험 보고서 - v8 고도화 피처 엔지니어링 및 정규화 스태킹

이 문서는 `v5` 정밀 결측 대치를 기반으로 4대 극단 예후/임상 시너지 피처를 새로 설계하여 주입하고, 베이스 모델 예측 간 다중공선성을 극복하기 위해 L2 규제를 주입한 Logistic Regression Stacking 구조를 구현한 `v8` 실험 보고서입니다.

---

## 1. 구현 사양 및 피처 설계 (Implementation & Features)
* **목적**: 예측의 임계 한계치(0.74 AUC)를 돌파하고 0.8 ROC-AUC에 수렴하기 위해, 타겟 분포와 비선형 시너지를 일으키는 고품질 파생 피처들을 대거 주입하고 메타 모델을 규제화함.
* **폴더 경로**: `src/v8/`
* **제출 산출물**: `src/v8/submission_v8_advanced.csv`

### 1.1 특정 시술 조합 예후 플래그 (`is_extreme_low_risk_procedure` / `is_high_prognosis_procedure`)
- **도메인 및 데이터 인사이트**:
  - `특정 시술 유형` 컬럼 분석 결과, `ICSI:ICSI`(성공률 0.99%) 및 `IVF:IVF`(성공률 1.04%) 조합은 난임 성공률이 1% 이하로 극도로 수렴하는 저예후군 집단입니다.
  - 반면, `ICSI / BLASTOCYST`(성공률 35.6%) 및 `IVF / BLASTOCYST`(성공률 36.6%) 등은 전체 평균인 25.8%를 크게 상회하는 고예후군 집단입니다.
  - 이 특이한 도메인 분류를 모델이 명확히 인식할 수 있도록 바이너리 형태로 제공하여 극단적인 결정 경계를 선명하게 학습하도록 도왔습니다.

### 1.2 고령 환자의 기증 난자 극복 시너지 플래그 (`elderly_donor_mismatch_synergy`)
- **데이터 인사이트**:
  - `시술 당시 나이`가 만 45-50세인 최고령 환자군의 성공률(16.7%)이 만 43-44세(11.8%)보다 비정상적으로 높게 측정됩니다.
  - 이는 임산부 본인의 노화된 난자 대신, 젊고 건강한 난자를 기증 공여(`난자 출처` == '기증 제공')받아 신체적 나이 한계를 성공적으로 우회한 시술 케이스가 존재하기 때문입니다.
  - 따라서 **(만 40세 이상 고령 환자) & (기증 난자 사용) & (기증자 나이가 만 30세 이하)**의 조건을 충족하는 고예후 시너지 환자 그룹을 식별하는 파생 변수를 추가했습니다.

### 1.3 PGS / PGD 유전 검사 정보의 의미 보존 플래그 (`is_pgs_tested` / `is_pgd_tested`)
- **데이터 인사이트**:
  - `착상 전 유전 검사 사용 여부`(PGS) 및 `착상 전 유전 진단 사용 여부`(PGD) 변수는 결측률이 98%를 상회할 정도로 극도로 낮습니다.
  - 하지만 통계 분석 결과, 검사 값 `1.0`을 기록한 집단의 성공률(14.8%)은 일반적인 환자군 평균(25%대)보다 확연히 낮습니다. 이는 반복 착상 실패나 유전적 취약성 등 "염색체 스크리닝이 필수적으로 동반되어야 할 정도로 기저 난임 상태가 심각한 고위험군 환자"만이 해당 검사를 거침을 뜻합니다.
  - 따라서 단순 평균 대치 시 발생하는 정보의 왜곡을 방지하고, 유전 검사를 통과해야 했던 중증 기저 난임 여부 자체를 보존하는 바이너리 플래그를 추출했습니다.

### 1.4 수정률 및 이식 효율 지표 다각화 (`icsi_fertilization_rate` / `embryo_transfer_efficiency`)
- **피처 설계**:
  - `icsi_fertilization_rate` = `미세주입에서 생성된 배아 수` / (`미세주입된 난자 수` + 1) (수정 성공률)
  - `embryo_transfer_efficiency` = `이식된 배아 수` / (`저장된 배아 수` + 1) (실제 이식 효율성)

## 2. 모델 및 앙상블 아키텍처 (Model & Stacking)

1. **5대 이종 베이스 분류기 학습 (Stratified 5-Fold)**:
   - **LightGBM A**: 트리 용량 극대화 (num_leaves: 63, max_depth: 9)
   - **LightGBM B**: 일반화 중심 소형 트리 (num_leaves: 31, max_depth: 6)
   - **XGBoost**: Pandas 카테고리 데이터 최적 학습 + L1/L2 강력한 규제 적용
   - **CatBoost**: 범주형 피처 특화
   - **AdaBoost**: DecisionTree 기반 부스팅을 통한 개별 트리와 다른 보조 학습
2. **Meta-Learner 개선 (L2-Regularized Logistic Regression)**:
   - 각 베이스 모델들이 예측한 5개의 Out-Of-Fold(OOF) 확률 예측 값들은 강한 다중공선성(Multicollinearity)을 띱니다.
   - 기존의 LightGBM 메타 모델은 과적합의 가능성이 높아, 스태킹 오차가 누적될 수 있습니다.
   - `v8`에서는 메타 모델로 강력한 L2 정규화가 탑재된 **`LogisticRegression(C=0.1)`**을 채택하여, 베이스 예측 확률값 간의 불필요한 과도 학습을 규제하고 안전한 소프트 가중 결합(Soft Calibration Stacking)이 이루어지도록 설계하였습니다.

## 3. 최종 결론
- v8은 HFEA 데이터에 특화된 4가지 고도화 임상 도메인 피처들과, 스태킹 시 흔히 발생하는 과적합을 방지하기 위한 정규화 메타 분류기(Logistic Regression)를 탑재하였습니다.
- 이를 통해 모델은 시술 세부 코드 내 숨겨진 저/고예후 군집의 경계를 매우 정밀하게 예측할 수 있으며, 0.74의 스코어 한계를 넘어서서 0.8 이상의 ROC-AUC를 향해 나아갈 수 있는 가장 견고한 파이프라인을 구축하였습니다.
