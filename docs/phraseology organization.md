# 난임 시술 도메인 용어 정리

이 문서는 `infertility_classification` 프로젝트의 데이터 컬럼들을 이해하기 위한 의학 용어 정리입니다.

## 1. 시술 종류

| 용어 | 의미 |
|---|---|
| **IVF** (In Vitro Fertilization, 체외수정) | 난자와 정자를 몸 밖(실험실)에서 인공적으로 수정시켜 배아를 만든 다음, 그 배아를 자궁에 이식하는 시술. 흔히 말하는 "시험관 아기" 시술 |
| **DI** (Donor Insemination / 인공수정) | 정자를 자궁이나 자궁경부에 직접 주입하는 시술. 체외에서 배아를 만들지 않음. IVF보다 간단하지만 성공률이 낮음 (데이터 기준 IVF 26.2% vs DI 12.9%) |

두 시술의 가장 큰 차이는 **"배아를 몸 밖에서 만드는가"** 입니다. IVF는 만들고, DI는 만들지 않습니다. 그래서 데이터에서 배아·난자 수치 컬럼들이 DI 시술자 전원(6,291건)에서 결측으로 나타나는 이유가 바로 이것입니다 — DI는 그 단계 자체가 없기 때문입니다.

## 2. IVF 시술 단계 개요

IVF는 아래 순서로 진행되며, 데이터 컬럼들도 대체로 이 순서를 따라 나열되어 있습니다.

1. 난자/정자 채취
2. 수정 (혼합)
3. 배아 배양
4. 신선 이식 또는 동결 보관 후 이식
5. 자궁에 배아 이식 → 임신 성공 여부 결정

## 3. 단계별 컬럼 용어 사전

### 3-1. 난자/정자 채취

| 컬럼 | 의미 |
|---|---|
| 배란 자극 여부 / 배란 유도 유형 | 보통 난자는 한 달에 1개만 나오는데, 호르몬 주사로 여러 개의 난자가 동시에 자라도록 자극하는 것. 한 번 시술로 여러 개의 배아를 만들기 위해 사용 |
| 수집된 신선 난자 수 | 채취 당시 갓 얻은 난자 개수 |
| 난자 출처 / 정자 출처 | 본인 제공 vs 기증 제공(타인의 난자·정자를 쓰는 경우) |
| 난자 채취 경과일 | 배란 자극 시작부터 난자를 채취하기까지 걸린 날짜 |

### 3-2. 수정 (혼합)

| 컬럼 | 의미 |
|---|---|
| 혼합된 난자 수 | 정자와 섞인(수정 시도한) 난자 개수 |
| 미세주입 (ICSI, Intracytoplasmic Sperm Injection) | 정자를 난자에 자연스럽게 섞는 대신, 가느다란 주삿바늘로 정자 1개를 난자 안에 직접 주입하는 기술. 정자 운동성이 약하거나 남성 요인이 있을 때 주로 사용 |
| 미세주입된 난자 수 / 미세주입에서 생성된 배아 수 | ICSI를 적용한 난자 개수, 그 결과 만들어진 배아 개수 |

### 3-3. 배아 배양

| 컬럼 | 의미 |
|---|---|
| 총 생성 배아 수 | 수정에 성공해서 배아로 자란 개수 |
| 배아 이식 경과일 | 배아를 며칠간 배양했는지. 보통 3일째(초기 분할기) 또는 5일째(배반포, blastocyst) 이식을 선택. 데이터상 5일째 이식이 성공률이 가장 높았음 |
| 착상 전 유전 검사 (PGT) / PGD / PGS | 이식 전에 배아의 유전자를 검사하는 것. **PGS** (Preimplantation Genetic Screening)는 염색체 수 이상 검사, **PGD** (Preimplantation Genetic Diagnosis)는 특정 유전질환 여부 검사 |

### 3-4. 신선 vs 동결 이식

| 컬럼 | 의미 |
|---|---|
| 신선 배아 사용 여부 | 배양한 배아를 그 주기에 바로 이식 |
| 동결 배아 사용 여부 | 배아를 일단 얼려서(동결, freezing) 보관했다가, 나중에(다음 생리 주기 등) 해동해서 이식하는 방식. 자궁 상태가 안 좋을 때 등 이식 시기를 늦추기 위해 사용 |
| 해동된 배아 수 / 배아 해동 경과일 | 동결했던 배아를 녹인 개수와 그 시점 |
| 기증 배아 사용 여부 | 본인의 배아가 아니라 다른 사람이 기증한 배아를 이식하는 경우 |
| 대리모 여부 | 본인이 아닌 대리모의 자궁에 배아를 이식하는 경우 |

### 3-5. 이식 및 결과

| 컬럼 | 의미 |
|---|---|
| 이식된 배아 수 | 자궁에 실제로 넣은 배아 개수 (1개만 넣으면 단일 배아 이식) |
| 단일 배아 이식 여부 | 배아를 1개만 이식했는지 여부 (여러 개 이식하면 다태아 임신 위험 증가) |
| 임신 성공 여부 | 최종 타겟 컬럼. 착상되어 임신이 확인됐는지 |

## 4. 불임 원인 분류 용어

| 용어 | 의미 |
|---|---|
| 주 불임 원인 / 부 불임 원인 | "주"는 가장 핵심적인 원인, "부"는 부수적으로 같이 있는 원인. 한 사람에게 원인이 여러 개 있을 수 있어 주/부로 구분해 기록 |
| 난관 질환 | 나팔관(난자가 자궁으로 이동하는 통로)이 막히거나 손상된 경우 |
| 배란 장애 | 난자가 정상적으로 배출되지 않는 문제 |
| 자궁내막증 | 자궁 안쪽 조직이 다른 곳에 자라나는 질환. 불임의 흔한 원인 중 하나 |
| 정자 농도 / 운동성 / 형태 | 정자 검사(정액검사)의 3대 지표. 정자가 너무 적거나(농도), 잘 움직이지 못하거나(운동성), 모양이 비정상적인(형태) 경우 |

## 5. 시술 횟수 관련 용어

| 용어 | 의미 |
|---|---|
| 클리닉 내 총 시술 횟수 | 지금 다니는 그 병원에서 받은 시술 횟수 (다른 병원 경력 제외) |
| 총 시술 횟수 | 다른 병원 경력까지 합친 전체 시술 횟수 |
